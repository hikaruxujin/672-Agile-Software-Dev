JSON Retrieves the JSON representation of the graph and displays it. Need to remove FreeTransform's shapes.

CREATE Select your shape from the dropdown and click Create. The shape will be created and dropped into the middle of the canvas.

SET TEXT Select a shape, then click Set Text. A prompt will open up where you can edit the text of the selected shape. Shape will resize if text is too large. Only resizes on rects. Doesn't clear when set to empty.

CONNECT Select a shape, then click Connect. You will enter the connection mode, where any shape you click will be connected to the selected shape. Click Connect again to quit the connection mode. The script checks for duplicate edges. Should only redraw affected edges.
